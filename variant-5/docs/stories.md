# Stories Page

## Purpose

Show stories as movement moments.

Not case studies.
Not CSR reports.
Not quiet documentary storytelling.

Stories should feel alive, bold and community-powered.

## StoryGalleryV5

Collage-style image section.

High energy.

Use multiple existing assets.

---

## Design Rules

Use V5 homepage visual language:

- bold uppercase typography
- orange and teal accents
- dark navy sections
- asymmetrical layouts
- collage images
- campaign energy

Avoid V4 report style.
Avoid V2 soft editorial style.
